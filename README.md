## tiny-search-engine

### 课题名称

微型搜索引擎（基于关键词的文本排序检索系统）

### 课题介绍
对于给定文本库，用户提交检索关键词，在文本库中查询与检索关键词最相关的 k 个文本（例如 k=5），并根据文本与检索关键词的相关度，对这 k 个文本进行排序，将排序后的结果返回给用户。

### 课题要求

（1）利用 TF-IDF 模型，为文本库中的文本创建索引（如倒排索引）

（2）用户输入的关键词可以是一个或多个

（3）对于返回的结果文本，需同时显示各检索关键词在结果文本中的出现 频度信息

（4）系统内支持返回结果文本的查看

### 主要算法

TF-IDF
